AVL Tree

AVL tree are the Alternatives or the better version of the binary search tree, because it remove the problem of Skewing in BST

Let's understand how...
suppose you have a empty BST and you have a sequence of input (4 - 3 - 2 - 1) to be pushed into the BST
let's push each of them into the tree
with the BST property the input should be inserted in sorting manner just like below-

To overcome this problem the avl tree comes with the self balancing property it help the tree to balance on both side with the help of balancing factor. now the question arises that what's this balancing factor ---- balancing factor is just a number determind by this formula-

Balancing Factor = HeightOf(leftSubTree) - HeightOf(rightSubtree)

If the balancing factor of any node in the tree is between (-1 , 1) then the tree is called balanced.

Rotations

1. LL Rotation

4. RL Rotation
